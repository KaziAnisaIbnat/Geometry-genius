How do you select elements from DOM?

Find the HTML element in 4 ways-
(i)Using getElementById: var element = document.getElementById("");
(ii)Using getElementsByClassName: var elements = document.getElementsByClassName("");
(iii)Using getElementsByTagName: var elements = document.getElementsByTagName("");
(iv)Using querySelector: var element = document.querySelector("");
Using querySelectorAll: var let elements = document.querySelectorAll("");

What Is A Callback Function?

A callback function is a function in JavaScript that is called by the first function after the first function has completed its task.
